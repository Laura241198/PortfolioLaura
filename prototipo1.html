Mis Prototipos de Apps

APP Presencia

📱 Versión Móvil

Este es el prototipo para la versión móvil de la app de presencia.

🔐 Al iniciar, verás una pantalla de inicio de sesión. No se necesitan credenciales, simplemente continúa para simular el acceso.

📲 Una vez dentro, navega usando el menú lateral hacia las diferentes secciones disponibles: fichajes, calendario, permisos, etc.

Explora cómo funcionaría la app desde tu móvil. ¡Es totalmente navegable!

💻 Versión Ordenador

Este prototipo corresponde a la versión web desktop de la aplicación.

🔐 También comienza simulando un inicio de sesión (sin necesidad de usuario o contraseña).

🖱️ Una vez dentro, puedes interactuar con el menú y acceder a las distintas pantallas, como lo harías desde un ordenador en la oficina.

Es ideal para visualizar cómo se adapta el sistema a pantallas grandes, con navegación fluida e intuitiva.
